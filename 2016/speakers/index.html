Talvez você não precise de um framework

Complexity: Advanced

JavaScript amadurece e ganha mais visibilidade a cada dia. A principal razão disso é a facilidade em desenvolver aplicações modernas usando algum dos vários novos frameworks que existem. Por outro lado, incluir um framework na sua tech stack cria uma dependência que pode custar caro no futuro e comprometer a saúde do seu codebase. É possível desenvolver JS sem ficar refém do React nem se apavorar com a atualização do Angular. Antes de você gritar "blasfêmia!", veja um caso real onde criamos uma interface rápida, fácil de manter e de testar usando boas práticas, Jasmine e algum jQuery.

Sane Asynchronous Programming with RxJS

Complexity: Intermediate

Programação Assíncrona é difícil por natureza. Controlar o estado de uma aplicação moderna que consome dados de diferentes fontes ao mesmo tempo pode se tornar um problema grande se não for feito de uma maneira sistemática e modular. Essa palestra apresenta o conceito de Programação Reativa usando a biblioteca RxJS. Você vai aprender o que são Observables, que tipo de operações essa primitiva suporta e como ela pode ser utilizada para estruturar comunicações assíncronas em um programa.
